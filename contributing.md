# Contributing

Have a framework to add? Submit a pull request with:

- The canonical project, paper, or article link.
- A concise one- or two-sentence description ending with a period.
- Placement in the most specific existing category.

Please run the local offline checker before opening a pull request:

```bash
python3 scripts/check_readme.py
```

If you have Node.js available, also run:

```bash
npx --yes awesome-lint
```
